# ابزارهای مورد نیاز

برای به دست آوردن همه فایل های با پسوند `txt` درون یک فولدر از `glob` استفاده کنید. نمونه کد زیر روش استفاده از `glob` را نشان می‌دهد:

```python
import glob

list_of_files = glob.glob('./files/*.txt')
for f in list_of_files:
    print(f)
```
